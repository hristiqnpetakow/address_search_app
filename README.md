Адресатор – Уеб приложение за търсене на адреси

¬¬ Описание

Адресатор е уеб приложение за търсене и визуализация на адреси, предназначено основно за потребители от България. Приложението използва ArcGIS Geocoding API за предоставяне на подсказки при въвеждане и за намиране на реални адресни резултати.

Проектът е реализиран с HTML, CSS и JavaScript, с цел кода да е ясен и лесен за поддръжка.

¬¬ Функционалности

- Търсене на адреси с live подсказки
- Поддръжка на клавиатура (Arrow Up / Down + Enter)
- Търсене с Enter без избор на подсказка
- Ограничаване на резултатите до България
- Премахване на дублиращи се резултати
- Показване на score (точност на съвпадение)
- Loading индикатор при заявки към API

¬¬ Използвани технологии

- HTML5 – структура
- CSS3 – стилове и responsive design
- JavaScript (ES Modules) – логика и капсулация
- ArcGIS Geocoding API – външна услуга за геокодиране

¬¬ ArcGIS API

~ Използвани endpoints:

- Suggest API – за подсказки при въвеждане
- FindAddressCandidates API – за реални адресни резултати

~ Основни параметри:

- text – въведеният адрес
- magicKey – идентификатор за прецизен резултат
- countryCode=BGR – ограничаване до България
- f=json – формат на отговора

~ Приложението поддържа:

- Избор на подсказка -> прецизни резултати
- Enter без избор -> всички възможни резултати

¬¬ Responsive Design

- Flexbox layout
- Media queries за мобилни устройства
- Оптимизирани фонови изображения за различни екрани
